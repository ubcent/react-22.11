# домашние задания

Сделан сайт\блог о здоровье на react с использованием webpack и библиотеки reactstrap.github.io.
Реализован layout, в который подрубаются все компоненты.

Для создание нового компонента необходимо выполнить команду npm run dev:c "Название компонента",будет создано 3 файла, index.js, css и jsx
Для запуска в разработческой среде используется команда npm run dev
Для запуска в продакт среде используется команда npm run build

3. частично реализована логика через новый слой, файлы которого находятся в папке containers (CommentListContainer.jsx и PostContainer.jsx)
4. созданы три новых компонента для отрисовки: Timer,CommentsList,Post 
5. в компоненте CommentsList задокументированы типы данных свойств нашего состояния через propTypes
